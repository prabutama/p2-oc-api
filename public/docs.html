Base URL

http://localhost:3001/api
Semua endpoint API menggunakan base URL di atas.

Customers Endpoints

GET /customers
Mendapatkan semua data customers
{ "success": true, "data": [ { "customerNumber": 103, "customerName": "Atelier graphique", "contactLastName": "Schmitt", "contactFirstName": "Carine", "phone": "40.32.2555", "addressLine1": "54, rue Royale", "city": "Nantes", "country": "France", "creditLimit": 21000.00 } ], "count": 122 }
GET /customers/total
Mendapatkan total jumlah customers
{ "success": true, "data": { "total_customers": 122 } }
GET /customers/fullname
Mendapatkan customers dengan nama lengkap (gabungan first name dan last name)
{ "success": true, "data": [ { "customerNumber": 103, "customerName": "Atelier graphique", "fullName": "Carine Schmitt", "phone": "40.32.2555", "city": "Nantes", "country": "France" } ] }
GET /customers/top-country
Mendapatkan negara dengan jumlah customers terbanyak
GET /customers/highest-credit
Mendapatkan customer dengan credit limit tertinggi
GET /customers/lowest-credit
Mendapatkan customer dengan credit limit terendah

Employees Endpoints

GET /employees
Mendapatkan semua data employees
GET /employees/total
Mendapatkan total jumlah employees

Offices Endpoints

GET /offices
Mendapatkan semua data offices
GET /offices/by-country
Mendapatkan jumlah offices per negara
{ "success": true, "data": [ { "country": "USA", "Total_Offices": 3 }, { "country": "France", "Total_Offices": 1 } ] }

Orders Endpoints

GET /orders
Mendapatkan semua data orders
GET /orders/total
Mendapatkan total jumlah orders
GET /orders/by-year-month
Mendapatkan orders berdasarkan tahun dan bulan
{ "success": true, "data": [ { "Year": 2003, "Month": 1, "Month_Name": "January", "Total_orders": 7, "Sum_Of_Orders": 7 } ] }
GET /orders/shipped
Mendapatkan jumlah orders dengan status 'Shipped'

Payments Endpoints

GET /payments
Mendapatkan semua data payments
GET /payments/total-amount
Mendapatkan total amount dari semua payments
{ "success": true, "data": { "Total_Amount": 8853839.23 } }
GET /payments/by-year
Mendapatkan payments berdasarkan tahun
{ "success": true, "data": [ { "Year": 2003, "Total_Amount": 1676851.95, "Sum_Of_Amount": 1676851.95 }, { "Year": 2004, "Total_Amount": 4313328.25, "Sum_Of_Amount": 5990180.20 } ] }

Products Endpoints

GET /products
Mendapatkan semua data products
GET /products/total
Mendapatkan total jumlah products
GET /products/by-line
Mendapatkan jumlah products per product line

Product Lines Endpoints

GET /productlines
Mendapatkan semua data product lines
GET /productlines/total
Mendapatkan total jumlah product lines

Health Check

GET /health
Mengecek status kesehatan server dan koneksi database
{ "status": "healthy", "database": "connected", "uptime": 3600.123, "timestamp": "2024-01-01T12:00:00.000Z", "environment": "development" }
